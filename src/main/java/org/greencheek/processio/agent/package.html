This package contains the JavaAgent that kicks of the process of monitoring /proc/PID/io
for the amount of IO that is being performed by the current JVM process. The PID is
obtain from the name of the RuntimeMXBean that is part of the MBeanServer.

By default the agent will periodically read the /proc/PID/io file for the current process. The
frequency at which it does this is {@value org.greencheek.processio.service.scheduler.ProcessIOScheduler#DEFAULT_FREQUENCY_IN_MILLIS}

The amount of io the process has done is then by default is stored as a MXBean within the local jvm's MBeanServer
The mbean is registered under the name {@value org.greencheek.processio.service.persistence.jmx.ProcessIOUsagePersistenceViaJmx#DEFAULT_COMPOSED_JMX_BEAN_NAME}

The defaults can be changed via options set on the -javaagent command, for example: 

       -javaagent:$CATALINA_HOME/lib/processio-0.0.1-SNAPSHOT-relocated-shade.jar=frequency=300000,jmxbeanname=io,jmxdomainname=my.domain

  • frequency : the time in millis that the process periodically records the io consumed by the application
  • jmxdomainname : the domain under which to register the bean
  • jmxbeanname : the name under which to register the bean
