Contains the MXBean that is stored in the JMX Server, to obtain the ProcessIO obtain from the JMX Service as an atomic unit. The MXBean makes it easy to obtain the bean via a proxy, and query for it's values: 

    ObjectName name = new ObjectName("org.greencheek:type=processiousage");
    ProcessIOUsageMXBean proxy = JMX.newMXBeanProxy(mbeanServer, name, ProcessIOUsageMXBean.class, false);
    double val = proxy.getSampleTimeKbPerSecondReadIO();
